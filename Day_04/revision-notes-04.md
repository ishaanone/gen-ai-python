# VS Code Shortcuts for Python 🚀

## 🧹 Format Code
- `Shift + Alt + F` → Format Python code

---

## ✏️ Line Editing

- `Alt + ↑ / ↓` → Move line up/down  
- `Ctrl + Enter` → Insert new line below  
- `Ctrl + Shift + Enter` → Insert new line above  
- `Ctrl + Shift + K` → Delete line  

---

## 🧭 Cursor Movement

- `Ctrl + →` → Jump word by word  
- `Ctrl + ↓` → Move cursor line down  

---

## ⚡ Quick Summary

| Shortcut               | Action             |
|------------------------|--------------------|
| `Shift + Alt + F`      | Format code        |
| `Alt + ↑ / ↓`          | Move line          |
| `Ctrl + →`             | Jump word          |
| `Ctrl + ↓`             | Move down          |
| `Ctrl + Enter`         | New line below     |
| `Ctrl + Shift + Enter` | New line above     |
| `Ctrl + Shift + K`     | Delete line        |

---

## 💡 Pro Tip

Use these 3 most frequently:

- `Ctrl + Enter` → Write faster  
- `Alt + ↑ / ↓` → Rearrange code  
- `Shift + Alt + F` → Clean code  

👉 These will make you **2x faster in coding**
